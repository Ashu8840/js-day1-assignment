# JavaScript Assignment – Day 2

This repository contains solutions to all 25 JavaScript questions from **Assignment 2 (Day-2)**.  
All programs are written in a single file named `assignment2.js`.  
Each program includes:
- A comment describing the question  
- Clean and simple JavaScript code  
- Console output written as a comment  

This makes it easy for anyone to understand and run the solutions.
